Quick Start

If you would like to quickly add Rich UI to your project. Head over to the GitHub and grab the build folder. Then Copy the Style and Script links bellow.

Some of our Components require JQuery read up on the individual componets to see if its necessary.

The link

Don't want to download the files? Just link it as below.
<link rel="stylesheet" href="https://rich-ui.netlify.app/Build/rich.min.css" />
